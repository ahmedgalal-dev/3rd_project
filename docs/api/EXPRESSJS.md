# Express API
## Installing & Running application
    1. install packages :
        - npm install
    2. running app :
        - npm run run-dev
## Dependencies
    1. Express js
    2. db-migrate
    3. db-migrate-pg
    4. dotenv
    5. pg
    6. cors
## Dev Dependencies
    1. Typescript
    2. ts-node-dev
    3. jasmine
    4. supertest
